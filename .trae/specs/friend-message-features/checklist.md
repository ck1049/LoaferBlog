# LoaferBlog - Friend and Message Features Verification Checklist

- [ ] User can search for other users by username
- [ ] User can send friend requests to found users
- [ ] User receives notifications for friend requests

## Message Read/Unread Status
- [ ] Messages have read/unread status tracking

- [ ] Red dot count matches total unread messages
