# LoaferBlog 功能更新 v2 - 产品需求文档

## Overview
- **Summary**: 本次更新包含聊天消息功能优化、添加好友功能实现、全局背景视觉效果优化以及帖子统计问题修复。
- **Purpose**: 提升用户体验，完善社交功能，修复现有统计数据问题。
- **Target Users**: LoaferBlog 平台的所有注册用户。

## Goals
- 优化聊天消息功能，支持文件下载、图片视频在线预览和保存，以及消息滚动逻辑优化
- 实现添加好友功能，支持通过用户名搜索用户并分页查询
- 优化全局背景视觉效果，增加炫酷的 CSS 动画
- 修复帖子收藏数统计问题，避免刷新页面归零
- 修复帖子浏览次数统计问题，避免刷新页面计数加 2

## Non-Goals (Out of Scope)
- 聊天消息的加密传输
- 添加好友的权限控制和验证机制
- 背景动画的性能优化
- 帖子统计数据的持久化存储优化

## Background & Context
- 现有聊天功能已实现基本的消息发送和展示，但文件处理和滚动逻辑存在问题
- 添加好友功能尚未完全实现，需要完善用户搜索和分页查询
- 全局背景视觉效果较为单调，需要增加动画效果提升用户体验
- 帖子收藏数和浏览次数统计存在 bug，影响用户体验

## Functional Requirements
- **FR-1**: 聊天消息文件支持下载功能
- **FR-2**: 聊天消息图片和视频支持在线预览和保存
- **FR-3**: 进入聊天窗口时默认将历史消息滑动到最底部（最新消息）
- **FR-4**: 发送消息时自动跟随滚动到底部
- **FR-5**: 实现添加好友功能，支持通过用户名搜索本站用户
- **FR-6**: 添加好友搜索结果支持分页查询，按照用户 ID 排序
- **FR-7**: 滑动到页面底部时显示"加载更多"按钮
- **FR-8**: 后端添加用户 ID 大于此参数的查询条件，避免深度分页问题
- **FR-9**: 全局背景视觉效果优化，增加炫酷的 CSS 动画
- **FR-10**: 修复帖子收藏数统计问题，避免刷新页面归零
- **FR-11**: 修复帖子浏览次数统计问题，避免刷新页面计数加 2

## Non-Functional Requirements
- **NFR-1**: 聊天消息滚动逻辑流畅，无卡顿
- **NFR-2**: 文件下载和预览功能响应迅速
- **NFR-3**: 添加好友搜索和分页查询性能良好
- **NFR-4**: 背景动画效果不影响页面加载速度和用户操作
- **NFR-5**: 帖子统计数据准确，无重复计数

## Constraints
- **Technical**: 现有技术栈和架构不变，基于 Spring Boot + Vue 3 实现
- **Business**: 本次更新需要在一周内完成
- **Dependencies**: 无外部依赖新增

## Assumptions
- 用户已登录系统，拥有有效的身份认证
- 系统已配置好文件存储和访问路径
- 数据库结构已支持所需的功能实现

## Acceptance Criteria

### AC-1: 聊天消息文件下载功能
- **Given**: 用户在聊天窗口中看到文件消息
- **When**: 用户点击文件链接
- **Then**: 系统开始下载该文件
- **Verification**: `programmatic`

### AC-2: 聊天消息图片视频预览和保存
- **Given**: 用户在聊天窗口中看到图片或视频消息
- **When**: 用户点击图片或视频
- **Then**: 系统显示图片或视频的预览，并提供保存选项
- **Verification**: `programmatic`

### AC-3: 聊天消息滚动逻辑优化
- **Given**: 用户进入聊天窗口或发送消息
- **When**: 历史消息加载完成或新消息发送成功
- **Then**: 聊天窗口自动滚动到最底部，显示最新消息
- **Verification**: `programmatic`

### AC-4: 添加好友功能实现
- **Given**: 用户进入添加好友页面
- **When**: 用户输入用户名并点击搜索
- **Then**: 系统显示匹配的用户列表，并支持分页加载
- **Verification**: `programmatic`

### AC-5: 全局背景视觉效果优化
- **Given**: 用户访问任何系统页面
- **When**: 页面加载完成
- **Then**: 系统显示带有炫酷 CSS 动画的背景效果
- **Verification**: `human-judgment`

### AC-6: 帖子收藏数统计修复
- **Given**: 用户收藏帖子后刷新页面
- **When**: 页面重新加载
- **Then**: 帖子收藏数保持不变，不会归零
- **Verification**: `programmatic`

### AC-7: 帖子浏览次数统计修复
- **Given**: 用户访问帖子详情页并刷新
- **When**: 页面重新加载
- **Then**: 帖子浏览次数只增加 1 次，不会重复计数
- **Verification**: `programmatic`
