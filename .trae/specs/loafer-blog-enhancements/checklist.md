- [x] 检查帖子卡片是否支持点击进入详情页

- [x] 检查管理员是否可以发布、编辑、删除帖子
- [x] 检查用户是否可以修改密码、注销账号

- [x] 检查用户是否可以查看收藏夹、浏览历史、点赞列表
- [x] 检查用户是否可以点赞帖子和评论
- [x] 检查消息系统是否支持聊天功能，包括联系人列表和聊天窗口
