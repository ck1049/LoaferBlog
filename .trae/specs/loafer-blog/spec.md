# LoaferBlog - 技术博客项目 - 产品需求文档

## Overview
- **Summary**: 一个基于Vue3+TypeScript+Vite前端和Java21+Spring Boot+MyBatis Plus+PostgreSQL后端的技术博客系统，支持用户登录注册、权限管理、公告管理、技术贴发布与搜索、评论回复、私信消息、分类管理、标签管理和敏感词过滤等功能。
- **Purpose**: 为技术人员提供一个分享和交流技术知识的平台，支持管理员进行内容管理和用户管理。
- **Target Users**: 技术开发者、技术爱好者、博客管理员。

## Goals
- 实现完整的用户认证系统（登录、注册、注销）
- 建立多级权限管理体系（管理员、普通用户、游客）
- 提供公告管理功能（查看、发布、编辑、删除）
- 支持技术贴的发布、搜索和查看
- 实现评论和回复功能
- 提供用户与管理员之间的私信消息功能
- 支持分类的动态管理（新增、编辑、删除）
- 支持标签的动态管理（新增、编辑、删除）
- 实现敏感词过滤系统

## Non-Goals (Out of Scope)
- 国际化支持
- 深色模式
- 多语言支持
- 第三方登录集成
- 内容推荐系统
- 后台管理系统的复杂数据分析功能
- 前端性能优化（如SSR、代码分割等高级优化）

## Background & Context
- 该项目是一个技术博客平台，旨在为技术人员提供一个分享和交流技术知识的空间。
- 技术栈选择考虑了现代前端和后端开发的最佳实践，确保系统的可维护性和扩展性。
- 权限管理和内容审核是系统的核心功能，需要确保管理员能够有效管理内容和用户。

## Functional Requirements
- **FR-1**: 用户认证系统
  - 支持用户注册、登录、注销功能
  - 支持密码重置功能
  - 支持JWT token认证

- **FR-2**: 权限管理系统
  - 实现管理员、普通用户、游客三级权限
  - 管理员拥有所有操作权限
  - 普通用户可评论、发送消息
  - 游客仅可查看内容

- **FR-3**: 公告管理
  - 管理员可发布、编辑、删除公告
  - 所有用户可查看公告

- **FR-4**: 技术贴管理
  - 管理员可发布技术贴
  - 所有用户可搜索、查看技术贴
  - 支持帖子的编辑和删除（仅限管理员）

- **FR-5**: 评论系统
  - 管理员和普通用户可评论帖子
  - 支持评论的回复功能
  - 管理员可删除恶意评论

- **FR-6**: 消息系统
  - 普通用户可向管理员发送消息
  - 管理员可查看和回复消息

- **FR-7**: 分类管理
  - 管理员可新增、编辑、删除分类
  - 一个帖子可属于多个分类
  - 删除分类前需移除该分类下所有帖子的分类信息

- **FR-8**: 标签管理
  - 管理员可新增、编辑、删除标签
  - 一个帖子可拥有多个标签
  - 删除标签前需移除该标签下所有帖子的标签信息

- **FR-9**: 敏感词过滤系统
  - 存储用户评论和消息的原文和过滤后结果
  - 页面展示时仅展示过滤后结果

## Non-Functional Requirements
- **NFR-1**: 性能要求
  - 页面加载时间不超过2秒
  - API响应时间不超过500ms

- **NFR-2**: 安全性
  - 实现SQL注入防护
  - 实现XSS攻击防护
  - 密码加密存储
  - JWT token安全管理

- **NFR-3**: 可扩展性
  - 系统架构支持水平扩展
  - 数据库设计支持未来功能扩展

- **NFR-4**: 可维护性
  - 代码结构清晰，遵循最佳实践
  - 完善的文档和注释

- **Business**:

## Acceptance Criteria

### AC-1: 用户认证功能
- **Given**: 用户访问登录页面
- **When**: 输入正确的用户名和密码
- **Then**: 登录成功并跳转到首页
- **Verification**: `programmatic`

### AC-2: 权限管理
- **Given**: 普通用户尝试访问管理员页面
- **When**: 点击管理员入口
- **Then**: 系统提示无权限访问
- **Verification**: `programmatic`

### AC-3: 公告管理
- **Given**: 管理员登录系统
- **When**: 发布新公告
- **Then**: 公告成功发布并在首页显示
- **Verification**: `programmatic`

### AC-4: 技术贴搜索
- **Given**: 用户在搜索框输入关键词
- **When**: 点击搜索按钮
- **Then**: 显示包含关键词的技术贴列表
- **Verification**: `programmatic`

### AC-5: 评论功能
- **Given**: 普通用户查看技术贴
- **When**: 输入评论内容并提交
- **Then**: 评论成功显示在帖子下方
- **Verification**: `programmatic`

### AC-6: 消息功能
- **Given**: 普通用户登录系统
- **When**: 发送消息给管理员
- **Then**: 消息成功发送并在管理员消息列表中显示
- **Verification**: `programmatic`

### AC-7: 分类管理
- **Given**: 管理员删除一个分类
- **When**: 确认删除操作
- **Then**: 分类被删除，该分类下的帖子移除该分类信息
- **Verification**: `programmatic`

### AC-8: 标签管理
- **Given**: 管理员删除一个标签
- **When**: 确认删除操作
- **Then**: 标签被删除，该标签下的帖子移除该标签信息
- **Verification**: `programmatic`

### AC-9: 敏感词过滤
- **Given**: 用户输入包含敏感词的评论
- **When**: 提交评论
- **Then**: 评论成功提交，存储原文和过滤后结果，页面显示过滤后结果
- **Verification**: `programmatic`
